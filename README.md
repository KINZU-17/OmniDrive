# 🚗 OmniDrive.co.ke

**Connecting you to the drive of your choice.**

OmniDrive is a high-performance, responsive vehicle marketplace that allows users to browse, filter, and compare vehicles from around the world. By integrating real-time currency conversion and nationality-based shipping logic, it provides a truly borderless shopping experience.

## 🚀 Core Features

- **Live Currency Conversion**: Integrated with the Open Exchange Rates API to provide real-time pricing in USD, EUR, JPY, KES, and more.

- **Intelligent Filtering**: Live "as-you-type" search combined with price sliders, category, condition, body style, fuel type, drivetrain, and color facets.

- **Advanced Comparison Engine**: A "Sticky Tray" interface allowing users to compare technical specs for up to 3 vehicles side-by-side.

- **Logistics Engine**: Dynamic shipping calculator that distinguishes between domestic and international freight, including automated import duty (tax) estimates.

- **Admin Inventory Management**: A secure dashboard to add, edit, or remove vehicles from the live database.

- **Multi-Payment Support**: MPesa, Credit/Debit Cards (Visa, Mastercard, AMEX), Bank Transfer

- **User Authentication**: Login/register with order tracking

- **Referral Program**: Earn $500 per successful referral

## 🛒 Product Categories

- **Cars**: From economy to luxury, 50+ models
- **Bikes**: Sports, touring, scooters, adventure
- **Buses**: City coaches, electric buses
- **Trucks**: Pickups, heavy duty
- **Vans**: Commercial, passenger

## 📂 Project Structure

The main interface consists of a Sticky Navbar for global actions and a Sidebar for specific vehicle attributes.

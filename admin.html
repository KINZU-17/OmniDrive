🚀 Deploy to Railway (Recommended — Free)

Railway gives you a live URL so you don't need ngrok anymore.

Step 1: Push your code to GitHub

git init git add . git commit -m "OmniDrive backend" git remote add origin https://github.com/YOUR_USERNAME/omnidrive.git git push -u origin main

Step 2: Go to railway.app → New Project → Deploy from GitHub → select your repo

Step 3: Add your environment variables in Railway dashboard (same as your .env file)

Step 4: Railway gives you a URL like https://omnidrive.up.railway.app — use that as your MPESA_CALLBACK_URL

🐳 Docker (Self-hosted)

docker build -t omnidrive . docker run -p 3000:3000 --env-file .env omnidrive

☁️ Render (Free tier)

Go to render.com → New Web Service → Connect GitHub → set Start Command to node server.js → add env vars → Deploy
